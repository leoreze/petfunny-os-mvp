# Entrega v0.1.4 — Ajustes finais de layout, menu e padrões premium

## O que foi feito
- Corrigido z-index do submenu do perfil para abrir sobre todos os componentes.
- Substituído o fundo azul escuro predominante do menu por uma paleta clara premium azul/rosa/grafite baseada na identidade da logo.
- Reforçado o post-it por baixo da logo horizontal, cobrindo toda a área da marca com sangria maior e menor transparência.
- Adicionados os menus Relatórios e Assistente IA.
- Padronizados inputs com atributos e máscaras visuais: WhatsApp, dinheiro, percentual, horário e CEP.
- Criado calendário premium com alternância visual entre Dia, Semana e Mês.
- Adicionadas margens entre linhas da tabela e menus contextuais em cards/listagens.
- Mantido menu desktop fixo e mobile sobreposto vindo da esquerda.

## Observação
Esta versão ainda é base visual. O calendário e os formulários são componentes simulados para definir o padrão oficial antes da v0.2 database.
