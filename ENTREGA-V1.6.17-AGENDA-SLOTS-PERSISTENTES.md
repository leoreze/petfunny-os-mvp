# FunnyOS v1.6.17 — Agenda Persistente + Cards Paralelos + Indicador de Ocupação

## Ajustes
- Slots configurados continuam visíveis mesmo quando estão lotados.
- Agendamentos do horário não desaparecem quando a capacidade é atingida.
- Cards no mesmo horário são renderizados lado a lado na visão diária.
- Indicador de ocupação mostra `usados/capacidade` e status visual de disponível/quase cheio/lotado.
- A disponibilidade para criar novo agendamento continua respeitando capacidade e horários passados.

## Banco
- Sem migration.

## Como rodar
```bash
npm start
```
